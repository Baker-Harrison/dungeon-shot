# Dungeon Shot

A browser twin-stick dungeon roguelite — clear locked rooms, pick in-run upgrades, beat the boss, spend meta currency between runs.

Built with **Phaser 3**, **TypeScript**, and **Vite**.

## Play locally

```bash
npm install
npm run dev
```

Open the URL Vite prints (default http://localhost:5173).

**Controls:** WASD move · mouse aim · click/hold to fire · `1` / `2` / `3` on upgrade pick

## Scripts

| Command | Description |
| --- | --- |
| `npm run dev` | Local game server |
| `npm test` | Dungeon / upgrade unit tests |
| `npm run build` | Production build to `dist/` |
